Project 1

Overview:
The Contact Management System project is a Python application designed to streamline the organization and management of contacts. It offers users a convenient platform to add, view, edit, delete, and search for contacts efficiently.

Key Features:
- Add Contacts: Users can easily add new contacts, including details such as name, phone number, email address, and notes.
- View Contacts: A user-friendly interface displays all contacts in a structured format, making it easy to browse and access contact information.
- Edit and Delete Contacts: Users have the flexibility to edit existing contacts or delete them as needed, ensuring their contact list remains up-to-date.
- Search Functionality: The system includes robust search capabilities, allowing users to quickly find specific contacts by name, phone number, or email address.
- Persistence: Contacts are stored persistently, ensuring that users can save their contact list and access it across multiple sessions.
- User-Friendly Interface: The application features an intuitive user interface, making it accessible to users of all technical levels.

Technologies Used:
Python: The project is implemented using the Python programming language, leveraging its simplicity and versatility.
